# BonAcheter

Lista de compras partilhada, orçamento e fluxos orientados ao Québec — aplicação **iOS** (SwiftUI).

## Estrutura do repositório

| Pasta | Conteúdo |
|--------|-----------|
| **`ios/`** | Projeto Xcode, código-fonte e [README detalhado](ios/README.md) |
| **`design/`** | Wireframes e assets de design |
| **`docs/`** | Notas de arquitetura (ex.: backend MVP) |

## Como começar

Abre o projeto no Xcode a partir da raiz do repositório:

```bash
open ios/BonAcheter.xcodeproj
```

Instruções completas (simulador, scripts, fluxo do app): **[ios/README.md](ios/README.md)**.
