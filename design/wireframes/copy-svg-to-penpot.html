Copier les wireframes SVG pour Penpot

Ouvrez cette page dans Chrome. Pour chaque écran, cliquez sur « Copier le SVG », puis dans Penpot (frame 375×812 sélectionné) faites Ctrl+V (ou Cmd+V). Le SVG est collé en vecteurs.

Si « Copier » ne fait rien : ouvrez ce dossier dans un serveur local (ex. python3 -m http.server 8080 dans ce dossier, puis http://localhost:8080/copy-svg-to-penpot.html). Sinon, ouvrez chaque fichier .svg dans un onglet, Ctrl+A puis Ctrl+C, et collez dans Penpot.
